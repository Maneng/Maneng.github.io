1. Overview

通常，当我们需要验证用户输入时，Spring MVC提供了标准的预定义验证器。

但是，当我们需要验证一个更具体的类型输入时，我们有可能创建自己的定制验证逻辑。

在本文中，我们将这样做 - 我们将创建一个自定义验证器来验证具有电话号码字段的表单，然后显示多个字段的自定义验证器.
